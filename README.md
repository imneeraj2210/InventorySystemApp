# Inventory Management System (ASP.NET Core MVC)

## 📌 Project Overview
A full-featured Inventory Management System built using ASP.NET Core MVC.
This application allows Admins to manage products, track orders, and manage users efficiently.

## 🚀 Features
- User Registration & Login
- Role-based Authorization (Admin/User)
- Add/Edit/Delete Products
- Order Management
- Admin Dashboard
- Secure Authentication

## 🛠️ Tech Stack
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap
- Razor Views
