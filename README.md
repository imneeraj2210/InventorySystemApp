# InventorySystemApp

ASP.NET Core MVC + PostgreSQL + Dapper based Inventory / Mini E-Commerce System.

## Features

### Customer Features
- Registration & Login
- Product Search
- Sort & Filter
- Pagination
- Add to Cart
- Checkout
- My Orders

### Admin Features
- Dashboard
- Product CRUD
- Order Management
- Website Settings
- Logo Upload

## Tech Stack
- ASP.NET Core MVC
- C#
- PostgreSQL
- Dapper
- Bootstrap
- Git & GitHub
